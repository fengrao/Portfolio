彗星爆破（Asteroids）

一款Arduino硬件上搭建和设计的射击游戏。
a shooting game runs on Arduino with a 8 x 8 LED metrix display

0 / 项目简介

一款Arduino硬件上搭建和设计的射击游戏。由Arduino Nano芯片、8*8的LED矩阵屏幕、加速规及按钮所搭建的硬件平台，并设计及编写可在该硬件上运行的射击游戏，通过C语言进行编写。如果对代码感兴趣可以在这里查看源码。
Asteroids is a shooting game runs on Arduino, where a player controls a spaceship hiting or avoiding asteriods falling above. It has a Arduino Nano microcontroller, a 8 x 8 LED metrix display, a potentiometer as input to control spaceship moving left and right and a tactile button used to shoot. This game is wriiten in C and whoever might be interested in source code please checkout here.
